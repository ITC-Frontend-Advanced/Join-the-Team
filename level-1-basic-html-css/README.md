# Basic Markup & Styling
**Goal:** Ensure they understand HTML structure and basic CSS.

## Task Ideas:
1. Build a simple personal profile page with:
    - Name, photo, short bio
    - List of hobbies
    - Footer with contact info

2. Style it using CSS:
    - Add background color, font styling, spacing (margin/padding)
    - Make a navigation bar with links
    - Use at least one hover effect

## What to Check:
- Correct HTML structure (semantic tags like <header>, <main>, <footer>)
- CSS syntax (`selectors`, `classes`, `IDs`)
- Ability to separate structure (HTML) and styling (CSS)

**Optional:** make it responsive using media queries.
