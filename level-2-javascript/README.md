# Basic JavaScript
**Goal:** Test JS fundamentals and DOM manipulation.

## Task Ideas:
1. Add a "like" button that increases a counter when clicked.
2. Create a simple to-do list:
    - Input field + button to add items
    - List updates dynamically in the DOM
    - Ability to remove items
3. Simple form validation:
    - Name and email fields
    - Alert if inputs are empty on submit

## What to Check:
- Variables, functions, events (`onclick`, `addEventListener`)
- DOM manipulation (`document.querySelector`, `appendChild`, `innerHTML`)
- Understanding of event handling
